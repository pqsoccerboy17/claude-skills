# Internal Communications Skill

Professional communication templates for business operations across Treehouse LLC, consulting, and Tap.

## Quick Start

When drafting communications, consider:
1. **Audience** - Tenant, client, investor, partner?
2. **Tone** - Formal, professional-friendly, casual?
3. **Purpose** - Inform, request, follow-up, escalate?
4. **Action needed** - Clear call-to-action?

## Communication Best Practices

### Email Structure

```
Subject: [Clear, specific, action-oriented]

[Greeting]

[Context/Background - 1-2 sentences]

[Main message/request]

[Specific details if needed]

[Clear call-to-action with deadline if applicable]

[Professional closing]
```

### Subject Line Patterns

| Type | Pattern | Example |
|------|---------|---------|
| Request | "Request: [specific item]" | "Request: Signed lease by Friday" |
| Update | "[Project]: [Status]" | "Tap Q4: Revenue exceeded target" |
| Action needed | "Action Required: [item]" | "Action Required: Maintenance access" |
| FYI | "FYI: [topic]" | "FYI: New parking policy effective Jan 1" |
| Follow-up | "Following up: [topic]" | "Following up: Contract discussion" |

### Tone Guidelines by Audience

**Tenants (Treehouse LLC)**
- Professional but approachable
- Clear on expectations and deadlines
- Empathetic when appropriate
- Always include contact info for questions

**Consulting Clients**
- Professional and confident
- Data-driven when possible
- Proactive about next steps
- Respectful of their time

**Investors (Tap)**
- Concise and metrics-focused
- Honest about challenges
- Clear on asks/support needed
- Forward-looking

## Quick Templates

### General Follow-Up

```
Subject: Following up: [Topic]

Hi [Name],

I wanted to follow up on our conversation about [topic] from [date/context].

[One sentence on current status or question]

Would you be available [specific time/date] to discuss? Alternatively, please let me know what works for your schedule.

Best regards,
[Your name]
```

### Meeting Summary

```
Subject: Summary: [Meeting Name] - [Date]

Hi all,

Thanks for joining today's discussion on [topic]. Here's a quick summary:

**Key Decisions:**
- [Decision 1]
- [Decision 2]

**Action Items:**
- [Person]: [Task] - Due [Date]
- [Person]: [Task] - Due [Date]

**Next Steps:**
[Brief description of what happens next]

Let me know if I missed anything or if you have questions.

Best,
[Name]
```

### Status Update

```
Subject: [Project/Property] Update - [Date/Period]

Hi [Name/Team],

Quick update on [project/property]:

**Progress:**
- [Accomplishment 1]
- [Accomplishment 2]

**In Progress:**
- [Current work item]

**Blockers/Needs:**
- [Any issues or support needed]

**Next Week:**
- [Planned activities]

Happy to discuss any questions.

[Name]
```

## Handling Difficult Communications

### Late Payment (Tenant)

```
Subject: Friendly Reminder: [Month] Rent Payment

Hi [Name],

I hope this message finds you well. I wanted to reach out regarding your [Month] rent payment, which was due on [date].

If you've already sent payment, please disregard this message. If there are any circumstances affecting your ability to pay, I'd appreciate the opportunity to discuss options.

Please remit payment by [new deadline] to avoid any late fees per your lease agreement.

Feel free to reach out if you have any questions.

Best regards,
[Name]
[Contact info]
```

### Scope Creep (Consulting)

```
Subject: Project Scope Discussion - [Project Name]

Hi [Name],

Thank you for the additional requirements you shared for [project]. I want to ensure we deliver exactly what you need while maintaining our timeline.

The new items you mentioned would expand our original scope:
- [New item 1]
- [New item 2]

I'd recommend we either:
1. Add these as a Phase 2 after we complete the current engagement
2. Adjust the current scope and timeline to accommodate

Would you have 15 minutes this week to discuss the best path forward?

Best,
[Name]
```

### Bad News (Investor Update)

```
Subject: Tap [Month] Update - Challenges and Path Forward

Hi [Investors],

This month's update includes some challenges I want to address transparently, along with our response.

**The Challenge:**
[Honest description of issue]

**Impact:**
[Quantified impact where possible]

**Our Response:**
[Specific actions being taken]

**Support Needed:**
[Any asks from investors]

**What's Going Well:**
[Balance with positive updates]

I'm available to discuss in more detail. Would welcome any guidance or connections that might help.

[Name]
```


### Calendar Integration

When scheduling, include:
- Clear meeting purpose in title
- Agenda in description
- Required prep in description
- Zoom/meet link

### Template Variables

Use consistent placeholders:
- `[NAME]` - Recipient name
- `[DATE]` - Relevant date
- `[PROPERTY]` - Property address
- `[AMOUNT]` - Dollar amount
- `[DEADLINE]` - Due date
