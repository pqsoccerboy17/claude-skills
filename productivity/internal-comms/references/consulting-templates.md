# Consulting Communication Templates

Professional templates for client engagement and project management.

## Engagement & Contracts

### Initial Engagement Letter

```
Subject: Engagement Letter - [Project Name]

Dear [Client Name],

Thank you for choosing [YourCo] Consulting. This letter confirms the terms of our engagement.

**Scope of Work:**
[Brief description of services]

**Deliverables:**
1. [Deliverable 1]
2. [Deliverable 2]
3. [Deliverable 3]

**Timeline:**
- Start date: [Date]
- Estimated completion: [Date]
- Key milestones: [List if applicable]

**Investment:**
- Total: $[amount]
- Payment schedule: [50% upfront, 50% on completion / monthly / etc.]
- Payment terms: Net [X] days

**What We Need From You:**
- [Access/information required]
- [Key contacts]
- [Decision-making process]

Please sign below and return with the initial payment to begin.

We're excited to work with you!

Best regards,
[Name]
[YourCo] Consulting
```

### Contract Renewal Discussion

```
Subject: Partnership Renewal - [Client Name]

Dear [Client Name],

**What We've Accomplished:**
- [Achievement 1]

- [Measurable outcome]

Based on our work together, I recommend:

**Options:**

2. **Project-based:** $[amount] for [specific scope]
3. **Custom:** Happy to discuss other arrangements

I'd love to schedule a call to discuss what makes sense for your needs.

Available times: [options]

Looking forward to continuing our work together.

Best,
[Name]
```

## Project Management

### Project Kickoff Summary

```
Subject: Kickoff Summary: [Project Name]

Hi [Client Name / Team],

Great kickoff meeting! Here's a summary of what we discussed:

**Project Overview:**
[2-3 sentence summary]

**Goals:**
1. [Primary goal]
2. [Secondary goal]

**Success Metrics:**
- [Metric 1]
- [Metric 2]

**Key Stakeholders:**
- [Name] - [Role]
- [Name] - [Role]

**Timeline:**
| Milestone | Target Date |
|-----------|-------------|
| [Phase 1] | [Date] |
| [Phase 2] | [Date] |
| [Final delivery] | [Date] |

**Next Steps:**
- [Your action item] - By [date]
- [Client action item] - By [date]

**Communication:**
- Weekly updates: [Day]
- Check-in calls: [Frequency]
- Questions: [Your email/phone]

Let me know if I missed anything.

[Name]
```

### Weekly Status Update

```
Subject: [Project Name] Weekly Update - Week of [Date]

Hi [Client Name],

Here's your weekly project update:

**Completed This Week:**
- [Task 1] ✓
- [Task 2] ✓

**In Progress:**
- [Task 3] - [X]% complete
- [Task 4] - On track for [date]

**Upcoming:**
- [Next week's focus]

**Blockers/Needs:**
- [Any issues requiring client input]
- [Or: No blockers - on track]

**Timeline Status:** [On track / At risk / Ahead]

**Budget Status:** [X]% of budget used, [Y]% of project complete

Happy to discuss any questions on our call [day].

[Name]
```

2. [Action 2]

### Change Request / Scope Discussion

```
Subject: Scope Discussion - [Project Name]

Hi [Client Name],

Thank you for the additional requirements you shared. I want to ensure we align on scope before proceeding.

**Original Scope:**
[Brief summary]

**New Requests:**
1. [New item 1]
2. [New item 2]

**Impact Assessment:**
- Timeline: [Additional X weeks]
- Investment: [Additional $X]
- Resources: [Any changes]

**Options:**
1. **Add to current project:** [New timeline/cost]
2. **Phase 2 scope:** Complete current, then address in next phase
3. **Trade-off:** Replace [existing item] with [new item]

I recommend [option] because [reasoning].

Can we discuss on [date/time]?

[Name]
```
